TelUGreenTrack
A web that uses Laravel and PHP using the MySQL database.
This web features: Dashboard Admin to controll; Schedules, Users, Emission. 
                    A welcome page. 
                    Authentication. 
                    And user views page with real-time data.
                    
How to run:

- Make a new folder
- Git clone this repo
- Composer Install
- run the laragon/xampp
- php artisan migrate
- php artisan db:seed
- php artisan serve
- click the link
done
